# Roman Calculator

> Provides calculators for converting an arabic number
to a roman glyph and vice versa

## Installation

```shell
go get github.com/derailed/imhotep/golabs/roman
```

## Usage

### Converts Arabic To Roman

```go
ArabicToRoman(10) // => X
```  

### Converts Roman To Arabic

```go
RomanToArabic("X") // => 10
```
