# Matrix

## Lab

* Write a program to compute 3x3 matrix addition and multiplication
* Write associated tests for these computations
* Extra credit: support substract and divide operators
* Extra credit: print out the matrices in a nice formatted output

## Expectations

```
1   1   1      3   3   3      4   4   4                                                                              
2   2   2  +   4   4   4  =   6   6   6                                                                              
3   3   3      5   5   5      8   8   8                                                                              
                                                                                                                    
1   1   1      3   3   3     -2  -2  -2                                                                              
2   2   2  -   4   4   4  =  -2  -2  -2                                                                              
3   3   3      5   5   5     -2  -2  -2                                                                              
                                                                                                                    
1   1   1      3   3   3     12  12  12                                                                              
2   2   2  *   4   4   4  =  24  24  24                                                                              
3   3   3      5   5   5     36  36  36                                                                              
                                                                                                                    
1   1   1      3   3   3      0   0   0                                                                              
2   2   2  /   4   4   4  =   0   0   0                                                                              
3   3   3      5   5   5      1   1   1 
```
